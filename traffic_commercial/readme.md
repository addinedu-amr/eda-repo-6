# 서울 버스 정류장별 승하차수 
- 서울 버스 정류장별 버스노선의 수
- 서울 버스 정류장별 위도 경도 (누락된 경우가 있을 수 있음) 를 하나의 파일로 정리함

# 서울 지하철역별 승하차수
- 서울 지하철역별 위도 경도를 하나의 파일로 정리함

# 도로별 평균속도 데이터와 위치정보
